# Sound Sources

Downloaded on 2026-05-01 for local use in Rogue Miners.

Most files are from OpenGameArt direct downloads. Where an exact sound was not available as a ready-made MP3, a close substitute was used and the existing procedural Web Audio layer still adds extra detail in-game.

## Notes

- Check each OpenGameArt page before redistribution. I selected files from pages that expose CC0 or permissive OpenGameArt licensing, but keep this file as a provenance note.
- The game still synthesizes some details procedurally, such as dice settle, gem pitch by color, night tension, owl/wolf/cricket accents, footsteps, and layered coin timing.
